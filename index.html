Hexo安装

1.新建文件夹，右键Git Bash here
2.输入（安装
`npm install -g hexo --registry=https://registry.npm.taobao.org ` 
3.输入（检验安装成功
`hexo -v`
4.输入（初始化文件夹
`hexo init --registry=https://registry.npm.taobao.org`
5.输入（安装组件
`npm install --registry=https://registry.npm.taobao.org`

6.输入 (检验成功

`hexo g`

`hexo s`

浏览器打开链接 http://localhost:4000/

7.输入

`npm install --save hexo-deployer-git`

NexT主题安装

Git Bash here输入
git clone https://github.com/theme-next/hexo-theme-next themes/next

GitHub与本地连接

1.注册GitHub
2.新建new repository:账号.github.io
勾选Initialize this repository with a README
3.用Sublime打开文件夹下的_config.yml文件
theme改为next
最后一行改为
deploy: type: git repository: https://github.com/godweiyang/godweiyang.github.io branch: master
4.右键打开git bash，输入：
git config --global user.name "账号"
git config --global user.email "邮箱"
输入
ssh-keygen -t rsa -C "邮箱"
输入
cat ~/.ssh/id_rsa.pub
5.打开GitHub设置新建SSH keys
将gitbash框中的ssh-rsa一直到.com复制到GitHub的key 框中，完成新建
6.回到Git bash
输入ssh -T git@github.com看到用户名即为成功
7.本地链接到GitHub
输入
hexo clean
hexo g
hexo d
浏览器输入账号.github.io
即可看到博客
